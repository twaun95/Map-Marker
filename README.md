# Map-Marker
카카오맵 api 를 사용하여 현재위치를 중심으로 한 지도와 검색을 통해 지도에 마커를 찍고 이동할 수 있다.  


- Clean Architecture + MultiModule + MVVM
- Coroutine, Flow
- JetPack
  - Data binding
  - DataStore
  - Paging3
- Retrofit, Glide, Timber, LeakCanary
- RecyclerView(ListAdapter), WebView(url 클릭시 이동)
- kakao map api
  - 장소 검색
    - 키워드
    - 카테고리
  - 주소 검색
